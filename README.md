# Вопросы и ответы по моделированию и проектированию ИС

## Стандарты IDEF и моделирование

- [x] **Какой стандарт из семейства IDEF используется для функционального моделирования?**
  - IDEF0

- [x] **Какой стандарт из семейства IDEF используется для процессного моделирования?**
  - IDEF3

- [x] **Какое назначение стандарта IDEF1X?**
  - Информационное моделирование (моделирование данных)

- [x] **Для чего используются диаграммы потоков данных (DFD)?**
  - Для моделирования процессов обработки данных

- [x] **Виды диаграмм, используемых в IDEF5**
  - Онто-графы

- [x] **Какое назначение стандарта IDEF3?**
  - Процессное моделирование

- [x] **Для чего используется стрелка в IDEF3 для соединения событий и состояний?**
  - Для описания отношений между событиями и состояниями

- [x] **Каково основное назначение стандарта IDEF5?**
  - Онтологическое моделирование

- [x] **Какие основные компоненты входят в модель процесса, созданную с использованием IDEF3?**
  - События, состояния, переходы

- [x] **Что означает термин "цикл" в IDEF3?**
  - Это последовательность событий, повторяющихся несколько раз

- [x] **Виды диаграмм, используемых в IDEF3**
  - Диаграммы переходов состояний
  - Диаграммы "что-если"

- [x] **Основные компоненты онтологической модели IDEF5**
  - Понятия, отношения, аксиомы

- [x] **На каком уровне абстракции находятся модели IDEF3?**
  - Концептуальный уровень

- [x] **На каком уровне абстракции находятся модели IDEF5?**
  - Концептуальный уровень

- [x] **Компоненты модели данных IDEF1X**
  - Сущности, связи, атрибуты

## UML и диаграммы

- [x] **Для чего предназначена диаграмма состояний в UML?**

- [x] **Какие виды диаграмм включены в UML?**

  - Диаграммы последовательности
  - Диаграммы классов

- [x] **Какая диаграмма используется для отображения статической структуры системы?**
  - Диаграмма классов

- [x] **Какая диаграмма показывает последовательность сообщений между объектами?**
  - Диаграмма последовательности

- [x] **Что такое UML?**
  - Язык моделирования

- [x] **Какой метод используется для описания функциональных требований к системе?**
  - UML-диаграммы

## Анализ требований и проектирование

- [x] **В чем заключается цель этапа анализа требований?**
  - Определение потребностей пользователей и бизнеса

- [x] **Почему важно учитывать качество информационной системы на этапе проектирования?**
  - Все варианты верны

- [x] **Что такое качество информационной системы?**
  - Степень соответствия системы предъявляемым требованиям

- [x] **Качество какой составляющей оценивается по простоте использования и интуитивности?**
  - Пользовательский интерфейс

- [x] **Какие факторы влияют на качество информационной системы?**
  - Время загрузки страниц
  - Безопасность
  - Удобство использования
  - Надежность
  - Доступность

- [x] **Какой аспект качества отражает способность системы сохранять работоспособность при сбоях?**
  - Надежность

- [x] **Какая метрика относится к производительности?**
  - Время отклика

- [x] **На каком этапе происходит документирование требований?**
  - На этапе анализа требований

- [x] **Основные задачи системного структурного анализа**
  - Описание существующего положения вещей в рассматриваемом бизнес-процессе
  - Предложение новых решений по структуре управления или технологии выполнения бизнес-процессов

- [x] **Тип системы с коэффициентом надежности 99,99%**
  - Высокой надежности

- [x] **Основные группы бизнес-процессов в организации**
  - Поддерживающие
  - Операционные
  - Стратегические

- [x] **Какие этапы включает процесс проектирования информационной системы?**
  - Анализ требований, разработка концепции, реализация, тестирование

- [x] **Какая диаграмма отражает реальное положение дел?**
  - модель "AS IS" (как есть)

- [x] **Кто является основным участником процесса проектирования информационной системы?**
  - Пользователи

- [x] **Может ли функционировать бизнес-процесс без участия человека?**
  - Может, автоматические бизнес-процессы функционируют без участия человека

- [x] **Что такое описание бизнес-процесса?**
  - Текстовое или графическое изображение алгоритма действий сотрудников компании

- [x] **Что такое информационная система?**
  - Комплекс программных и аппаратных средств, предназначенный для хранения, обработки и передачи информации

- [x] **Какая модель жизненного цикла предполагает последовательные этапы без возврата?**
  - Каскадная модель
